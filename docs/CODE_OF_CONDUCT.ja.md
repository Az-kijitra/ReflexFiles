# 行動規範

本プロジェクトは Contributor Covenant Code of Conduct に準拠します。

## 私たちの約束
年齢、体格、障害の有無、人種、民族、宗教、性別、性的指向などに関わらず、
すべての参加者が安心して参加できる環境づくりを約束します。

## 望ましい行動
- 相手への敬意を持つ
- 建設的なフィードバックを受け入れる
- コミュニティにとって最善の行動を選ぶ

## 望ましくない行動
- ハラスメント、差別的な発言
- 侮辱、攻撃的な言動
- 公的または私的な嫌がらせ

## 運用
不適切な行為を確認した場合、メンテナに報告できます。
報告内容は慎重に確認し、迅速かつ公平に対応します。

## 参照
本規範は Contributor Covenant 2.1 を基にしています。
https://www.contributor-covenant.org/version/2/1/code_of_conduct.html
