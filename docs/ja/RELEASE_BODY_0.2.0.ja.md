ReflexFiles v0.2.0 では、ビューアー統合と操作性改善を実施しました。

## 主な更新
- ReflexViewer 相当の表示フローを ReflexFiles 本体に統合。
- Markdown 表示で HTML/Text のトグルに対応。
- テキスト/ソース表示でシンタックスハイライト + 行番号表示に対応。
- 画像表示（PNG/JPEG/BMP）の読み込み安定性とズーム挙動を改善。
- インストーラーの言語選択（English/Japanese）と、言語連動のマニュアル/設定切り替えに対応。
