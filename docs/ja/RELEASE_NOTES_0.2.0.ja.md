# リリースノート v0.2.0 (2026-02-14)

## 概要
本リリースでは、ReflexViewer の機能を ReflexFiles に統合し、テキスト・Markdown・ソースコード・画像の閲覧体験を強化しました。

## 主な変更
- ReflexFiles 内にビューアーを統合（外部の別アプリ起動なしで閲覧可能）。
- Markdown 表示の強化:
  - HTML プレビュー表示
  - Text 表示（シンタックスハイライト対応）
  - ビューアー内トグルで HTML/Text 切り替え
- ソースコードの色分け表示を強化し、行番号表示を追加。
- 画像表示（PNG/JPEG/BMP）の改善:
  - 読み込み経路の安定化
  - 拡大縮小挙動の改善
  - 体感速度の改善
- ビューアー操作の改善:
  - キー操作の整理
  - スクロール/ズームの一貫性向上

## 多言語・インストーラー
- NSIS インストーラーで言語選択（English/Japanese）に対応。
- 言語選択に応じたマニュアル/設定ファイルの切り替えに対応。

## 品質・安定性
- 統合作業中に見つかったビューアー起動・表示の不安定動作を修正。
- 実運用フィードバックに基づく表示・操作性の調整を実施。
